# 灵渲科技官网开发计划

## 项目概述
本项目旨在为上海灵渲科技有限公司创建一个现代化的官方网站，核心理念为“数字创世纪”。网站将采用前沿的3D交互技术、AI驱动功能，并支持中英文双语，以展示公司的技术实力和创新精神。

## 技术栈
- **前端框架:** Next.js 15 + React 19
- **3D渲染:** React Three Fiber, Drei, Postprocessing
- **UI组件库:** shadcn-ui
- **样式:** Tailwind CSS v4
- **动画:** GSAP + Framer Motion
- **国际化:** next-intl
- **语言:** TypeScript
- **包管理器:** pnpm
